About GEOLOGIA

GEOLOGIA is a novel programming language that bridges the gap between Earth sciences and computer science. By modeling computational concepts after geological processes, GEOLOGIA creates an intuitive programming experience for those familiar with geology while teaching geological concepts to programmers.

Core Philosophy

  • Time-Based Execution: Programs evolve through geological time
  • Natural Transformations: Operations represent natural geological processes
  • Layered Structure: Data and code organized in geological strata
  • Educational Value: Teaching programming through familiar Earth science concepts

Key Features

Geological Terminology

Every keyword and operation has geological meaning, creating a natural connection between code and Earth processes.

Natural Data Types

Rock types (igneous, sedimentary, metamorphic) represent different data types, making data classification intuitive.

Transformation Metaphors

Operations mirror geological processes, with mathematical operations represented by processes like erosion, compression, and deposition.

Educational Synergy

Learn programming concepts and geological processes simultaneously through the language's dual-purpose design.

Visual Code Organization

Code is organized in strata, creating a visual representation of program flow that mirrors geological formations.

JavaScript-Based Interpreter

Easy to run in modern browsers with a web-based playground for experimentation and learning.

Code Examples

Hello World

#fossil# GEOLOGIA Hello World Program
#fossil# This program outputs "Hello, World!" to the surface (console)

expose "Hello, World!"

Sum Calculator

#fossil# Simple sum calculation program in GEOLOGIA
#fossil# This calculates the sum of numbers from 1 to 10

#fossil# Initialize the sum to 0
sediment sum igneous 0

#fossil# Add each number one at a time
sediment sum igneous sum deposit 1
sediment sum igneous sum deposit 2
...
sediment sum igneous sum deposit 10

#fossil# Show the final result
expose "Sum of numbers from 1 to 10 is:"
expose sum

Temperature Converter

#fossil# GEOLOGIA Temperature Converter
#fossil# Define a function to convert from Celsius to Fahrenheit

era celsius_to_fahrenheit
    #fossil# Get the temperature in Celsius (param0)
    sediment celsius igneous param0

    #fossil# Formula: F = C * 9/5 + 32
    sediment step1 igneous celsius compress 9
    sediment step2 igneous step1 dissolve 5
    sediment fahrenheit igneous step2 deposit 32

    #fossil# Return the result
    uplift fahrenheit
